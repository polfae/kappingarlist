# Kappingarklárt v4.8.2

Bug-fix version from 4.8.1.

Fixes in this version:

- Made adding tasks more reliable, especially when filters are active
- New tasks opened in the editor without re-rendering the checklist first
- Saving task edits is more defensive and keeps the task stable in its section
- Deadline time is now entered and saved as 24-hour HH:MM
- Time examples: 09:30, 14:00, 18:45
- Added safeguards around task editing so missing references do not break saving

Open `index.html` in your browser.
